# Blackjack

A simple Blackjack game implemented in JavaScript.

## Getting Started

Follow these steps to run the app locally and test it.

---

### **Running the App**

**Install Dependencies**  
 Make sure you have Node.js installed. Then, install the required dependencies:

```bash
npm install
```

**Run the App**

```bash
npm start
```

**Run the Tests**

```bash
npm test
```
